Services

How can I be of service?.

Web Development

Web development begins with a design document stating the General information, Background , Goals, Audience and Content for the site. A Visual Site Map is done to help aid the client in visualizing the site design. All Content images and text are all determined before the designing of the site is done. Client involvement from start to finish.


Website Maintenance

Web maintenance involves remaking a web site to be responsive in the design. Today all web sites need to be responsive to allow it to be viewed across various screen sizes of the mobile devices. Maintenance also includes updating information of text and images on display.


Branding

Web Branding involves getting the visitors to your site. Having the logo, brand styles, products or services on display. Being picked up by search engines and creating a social presence.
